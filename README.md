# dynamic-to-do-list-js

This project is a simple To-Do List application built using **HTML**, **CSS**, and **JavaScript**. It allows users to add, display, and remove tasks dynamically. The app also uses **Local Storage** to persist tasks across browser sessions.

## Features

* Add new tasks to the list
* Display tasks dynamically using DOM manipulation
* Remove tasks from the list
* Save and load tasks using Local Storage
* Support adding tasks by pressing the Enter key

## What I Learned

While building this project, I learned:

* How to use the `DOMContentLoaded` event to ensure JavaScript runs after the page loads
* How to select and manipulate DOM elements using JavaScript
* How to create elements dynamically with `document.createElement`
* How to attach event listeners to buttons and input fields
* How to update the DOM in response to user actions
* How to store, retrieve, and update data using Local Storage
* How to structure functions for adding, creating, and removing tasks
* How to maintain a synchronized tasks array between the UI and Local Storage

## Technologies Used

* HTML
* CSS
* JavaScript
* Local Storage

## How to Run

1. Download or clone the project files
2. Open `index.html` in a browser
3. Add tasks, remove tasks, refresh the page, and verify that tasks persist
